The primary goal of SolvingRT is to quantify exercise mechanics, to be able to standardize the technique or tempo used while doing a lift. This can be used to make experimental studies more rigorous â€” for instance by making sure that the velocity is kept the same while performing a movement in a research on mind-muscle connection. Also, it would reduce the cost, or increase the reliability of longitudinal studies by making it possible for subjects to film themselves while training and use the videos to, again, standardize the movement quality.

SolvingRT app

pip install git+https://github.com/samueleblanc/SolvingRT.git SolvingRT is not available on a cellphone since it is a Python library.


Code to produce this video:

import solvingrt.solve as srt

athlete1 = srt.Athlete(1.85, 100, 0.35, 10, "left")
exercise1 = srt.Exercise("Preacher curl", "biceps", "path/to/preacher_curl.mp4", athlete1, ["torque", "velocity lost"])

exercise1.video_resize(720, 400)
exercise1.angle_with_gravity()

exercise1.play_video()
